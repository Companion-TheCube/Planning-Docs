# Code of Conduct

All participants in the Companion-TheCube project are expected to abide by this Code of Conduct. The goal is a welcoming, respectful, and productive community.

## Our standards

Everyone participating in the project is expected to:

- Be respectful and courteous.
- Be inclusive of people from all backgrounds and experience levels.
- Be constructive and solution-oriented when giving feedback.
- Be transparent about decisions and actions that affect others.

## Unacceptable behavior

The following behaviors are not acceptable:

- Harassment, intimidation, or discrimination
- Abusive, insulting, or demeaning language
- Personal attacks or flame wars
- Public or private harassment
- Derogatory comments about protected characteristics
- Posting sexually explicit or violent content
- Trolling, baiting, or other disruptive activity
- Sharing private information without consent

## Enforcement

Project maintainers may take action including:

- Issuing a warning
- Removing offensive content
- Temporarily suspending access
- Permanently banning repeat or egregious offenders

Enforcement decisions are at the discretion of project maintainers.

## Reporting

If you experience or witness unacceptable behavior:

- Open a confidential GitHub issue tagged `private:conduct`, if that workflow is in place
- Email the maintainers at `conduct@companion-thecube.org`

Reports should be handled with discretion and respect for privacy.

## Acknowledgments

This Code of Conduct is adapted from widely used community guidelines intended to keep technical communities welcoming and harassment-free.
